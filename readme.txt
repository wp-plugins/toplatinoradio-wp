== Description ==

Add the Top Latino Radio Player as widget to your Wordpress site. Agrega el player de Top Latino Radio como widget en tu sitio de Wordpress.

== Installation ==

1. Upload the file 'toplatinoradio.php' to the '/wp-content/plugins/' directory .Sube el archivo toplatinoradio.php al directorio '/wp-content/plugins/'.
2. Activate the plugin through the 'Plugins' menu in WordPress. En el escritorio de tu WP ingresa a "Plugins"-"Plugins instalados" y activa "Top Latino Radio WP-Player".
3. The new widget is now in the widgets section. Encontraras el nuevo widget en la seccion Apariencia - Widgets.
